# 🛡️ MiniAV - Scanner de Malware Baseado em Assinaturas

Um antivírus **educacional e funcional** em Python, criado para ensinar lógica de detecção de malware via hashes MD5.

✅ Detecta arquivos infectados usando uma base de assinaturas.
✅ Varredura recursiva de diretórios.
✅ Relatório claro em terminal.
🚀 Ideal para portfólio em cibersegurança.
